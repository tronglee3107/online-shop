# 🛒 Online Shop - Shoppy

An e-commerce web application built with the MERN stack (MongoDB, Express.js, React, Node.js).

🔗 **Live Demo**: [https://shoppy-eta.vercel.app](https://shoppy-eta.vercel.app)

---

## 📦 Features

- ✅ User Authentication
- 🛍️ Product Management
- 🛒 Shopping Cart
- 📦 Order Processing
- 📱 Responsive Design
- 🛠️ [Admin Dashboard](https://shoppy-eta.vercel.app/seller)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### Hosting
 - Vercel
---
### Prerequisites
- Node.js (v20 or above)
- npm (v6 or above)
- MongoDB (local or cloud)
